## 讲解

这次主要是把 RecyclerView 比较常用的基本的点，在这里集中整理一下。从这篇文章主要梳理以下几点：

* 优雅的实现：item 点击事件 & item 长点击事件
* RecyclerView 添加 divider 的标准姿势
* RecyclerView 实现 item 的拖曳排序和滑动删除
* 拖曳排序时，限制首个 item 固定的实现
